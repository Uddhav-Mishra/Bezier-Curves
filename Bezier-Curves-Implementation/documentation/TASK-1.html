COMPUTER GRAPHICS

TASK-1 AND TASK-2

Task 1: Implement the de Castlejau algorithm for evaluating the entire 2D Bezier curve of degree n.

Task 2: Make your curve editable in the following sense: Addition of control Point: Every time we click on the canvas, a new point will be created and a new Bezier curve of appropriate degree (based on the number of points) will be redrawn.

Deletion of control point: We can delete an already existing control point and redraw the new Bezier curve of appropriate degree.

Control Point Movement: An user can drag any control point of the curve and correspondingly the curve should get update automatically

CRITICAL REGION OF POINT

To select a point for deleting it or moving it to a new position a critical region has been defined for each and every control point of the curve

The critical region of the point is small circle of radius 10 with the point as the center
